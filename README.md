# Scripts Repository
A personal collection of scripts built and used on the job for solving real-world problems across cloud infrastructure and endpoint management.
This repo serves as both a reference library for myself and a resource others can use or adapt.

# What's Inside

Cloud Infrastructure – Scripts for provisioning, configuration, and automation across Azure and Microsoft 365 environments.
Endpoint Management – Scripts for Intune, Autopilot, Defender, and general Windows/macOS device management tasks (detection, remediation, deployment, cleanup).
